# 🚀 SiberEko - Yeni Nesil Yazılımcı Ekosistemi

SiberEko, yazılım ve siber güvenlik meraklılarını bir araya getiren, React Native ve Expo ile geliştirilmiş yeni nesil, "Öğren-Kazan" (Learn-to-Earn) odaklı ve oyunlaştırılmış bir topluluk uygulamasıdır.

---

## 📌 Projenin Amacı ve Oyunlaştırma Özellikleri

Bu projenin temel amacı; yazılımcıların yeteneklerini sergileyebileceği bir topluluk havuzu oluştururken, aynı zamanda eğitim modülü ile teknik bilgilerini test edip ekosistem içinde prestij kazanabilecekleri dinamik bir yapı kurmaktır.

Proje, klasik bir listeleme uygulamasının ötesine geçerek **Oyunlaştırma (Gamification)** dinamikleri ile kullanıcıları sürekli öğrenmeye teşvik eder:

* 🎮 **Seviye ve XP Sistemi:**  
  Kullanıcılar eğitim modülünde doğru bildikleri her soru için **+300 XP** kazanır.  
  Her **1500 XP'de bir Level (Seviye)** atlayarak gelişimlerini somut şekilde görürler.

* 🎖️ **Dinamik Rozet Yönetimi:**  
  Kullanıcılar başarılarına göre özel rozetler kazanır:  
  - 🥇 "İlk Adım" → İlk doğru cevap  
  - 🏆 "Eko-Master" → 12+ doğru cevap  
  Kazanılan rozetler profilde kalıcı olarak sergilenir.

* 🔔 **Gerçek Zamanlı Senkronizasyon Döngüsü:**  
  Kullanıcılar:
  - Soru çözdükçe  
  - Profil bilgilerini güncelledikçe  

  Anında **XP, seviye ve rozet kazanımları** tüm topluluğa yansıtılır.  
  Böylece “Keşfet” ekranı sürekli canlı ve dinamik kalır.

## 🛠️ Nasıl Çalıştırılır? (Installation & Run)

Projeyi bilgisayarınızda yerel olarak çalıştırmak için aşağıdaki adımları takip edin:

**1. Projeyi Klonlayın:**
Terminali açın ve şu komutları çalıştırın:

```bash
git clone https://github.com/farukkemree/SiberEko.git
cd SiberEko

2. Gerekli Kütüphaneleri Kurun:

npm install

3. Uygulamayı Başlatın:

npx expo start -c

4. Test Edin:

📱 Fiziksel Cihazda:
Telefonunuza Expo Go uygulamasını indirin ve terminalde çıkan QR kodu okutun.

💻 Sanal Cihazda (Emulator):
Android Studio kuruluysa terminalde a tuşuna basarak emülatörde başlatabilirsiniz.
